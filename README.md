## Description

Node JS server for manage daily Habits.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```

## DB seed

```bash
$ npx prisma db seed
```

## DB ide (Prisma Studio)

```bash
$ npx prisma studio
```

## Generating ERD (Entity Relationship Diagram) 
https://www.npmjs.com/package/prisma-erd-generator

```bash
$ npx prisma generate
```
